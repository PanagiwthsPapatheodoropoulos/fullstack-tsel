# Required Setup
1. Node.js (v16 or higher)
2. MySQL Server (v8.0 or higher)
3. npm (Node Package Manager)


# Installation Commands
```bash
npm install
npm install bcrypt
npm install multer bcrypt
npm install express-session
npm install dotenv
```


# Start Application
```bash
npm start
```
